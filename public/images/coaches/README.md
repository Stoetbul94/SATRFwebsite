# Coach Images Directory

This directory contains images for the SATRF coaching services page.

## Required Images

### Coach Sarah van der Merwe
- **Filename**: `coach-sarah.jpg`
- **Recommended size**: 400x600px (portrait orientation)
- **Format**: JPG or PNG
- **Description**: Professional headshot of Coach Sarah van der Merwe

### Coach Michael Botha
- **Filename**: `coach-michael.jpg`
- **Recommended size**: 400x600px (portrait orientation)
- **Format**: JPG or PNG
- **Description**: Professional headshot of Coach Michael Botha

## Image Guidelines

1. **Quality**: Use high-resolution images (minimum 400x600px)
2. **Style**: Professional headshots with neutral backgrounds
3. **Format**: JPG or PNG format
4. **File size**: Optimize for web (under 500KB per image)
5. **Aspect ratio**: Portrait orientation (3:4 or 2:3 ratio)

## Adding Images

1. Place the coach images in this directory
2. Ensure filenames match exactly: `coach-sarah.jpg` and `coach-michael.jpg`
3. The coaching page will automatically display the images once they're added

## Fallback

If images are not available, the page will display placeholder icons with instructions to add the photos.

## Image Optimization

For best performance, consider:
- Compressing images using tools like TinyPNG or ImageOptim
- Using WebP format for modern browsers (with JPG fallback)
- Implementing lazy loading for better page performance
