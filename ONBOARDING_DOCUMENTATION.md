## Athlete Onboarding

### Getting Started

#### Step 1: Account Creation
1. **Visit the Website**
   - Go to [satrf.org](https://satrf.org)
   - Click "Register" in the top navigation

2. **Fill Registration Form**
   - **Personal Information:**
     - First Name: Your legal first name
     - Last Name: Your legal last name
     - Email: Use a valid email you check regularly
   
   - **Account Security:**
     - Password: Minimum 8 characters with uppercase, lowercase, and number
     - Confirm Password: Must match exactly
   
   - **Membership Details:**
     - Membership Type: Select your category
       - Junior: Under 18 years
       - Senior: 18-59 years
       - Veteran: 60+ years
     - Club: Your shooting club name

3. **Complete Registration**
   - Review all information
   - Accept terms and conditions
   - Click "Create Account"
   - Check email for verification (if enabled)

#### Step 2: First Login
1. **Access Login Page**
   - Go to [satrf.org/login](https://satrf.org/login)
   - Enter your email and password
   - Click "Sign In"

2. **Complete Profile**
   - Upload profile photo (optional)
   - Add contact information
   - Verify club affiliation
   - Set notification preferences

#### Step 3: Explore the Platform

##### Dashboard Overview
- **Quick Stats:** View your current rank, best score, and recent activity
- **Upcoming Events:** See events you're registered for
- **Recent Scores:** Your latest score submissions
- **Quick Actions:** Register for events, upload scores, view leaderboard

##### Navigation Menu
- **Home:** Return to dashboard
- **Events:** Browse and register for competitions
- **Scores:** Upload and manage your scores
- **Leaderboard:** View rankings and statistics
- **Profile:** Manage your account settings

### Core Features Guide

#### Event Registration
1. **Browse Events**
   - Go to "Events" in navigation
   - View upcoming competitions
   - Filter by location, type, or date
   - Click on event for details

2. **Register for Event**
   - Click "Register" button on event page
   - Review event details and requirements
   - Confirm registration
   - Receive confirmation email

3. **Manage Registrations**
   - View registered events in dashboard
   - Cancel registration (if allowed)
   - Download event information

#### Score Upload
1. **Access Score Upload**
   - Go to "Scores" → "Upload Score"
   - Or use "Upload Score" from dashboard

2. **Fill Score Form**
   - **Event Selection:** Choose the event you participated in
   - **Discipline:** Select your shooting discipline
   - **Score Details:**
     - Score: Your total score (0-600)
     - X-Count: Number of X-ring hits (0-60, optional)
     - Notes: Any relevant comments
   
   - **Documentation:**
     - Upload score sheet (PDF, JPG, PNG)
     - Maximum file size: 10MB
     - Drag and drop or click to browse

3. **Submit Score**
   - Review all information
   - Click "Submit Score"
   - Score status: "Pending Approval"
   - Receive confirmation

#### Viewing Leaderboards
1. **Access Leaderboards**
   - Go to "Leaderboard" in navigation
   - Choose leaderboard type:
     - Overall Rankings
     - Event-Specific
     - Club Rankings

2. **Filter Results**
   - Select discipline (e.g., 10m Air Rifle)
   - Choose category (junior/senior/veteran)
   - Set time period (all/year/month/week)

3. **Understand Rankings**
   - Position: Your rank
   - Best Score: Highest score achieved
   - Average: Mean score across events
   - X-Count: Total X-ring hits
   - Events: Number of competitions

### Best Practices

#### Score Management
- **Upload Promptly:** Submit scores within 48 hours of competition
- **Accurate Documentation:** Ensure score sheets are clear and legible
- **Complete Information:** Fill all required fields
- **Follow Up:** Check score status regularly

#### Event Participation
- **Early Registration:** Register well before deadlines
- **Read Requirements:** Review event rules and equipment needs
- **Stay Informed:** Check for updates or changes
- **Plan Ahead:** Consider travel and accommodation

#### Performance Tracking
- **Monitor Progress:** Check leaderboard regularly
- **Set Goals:** Use statistics to improve performance
- **Compare Categories:** Understand your ranking context
- **Track Trends:** Monitor improvement over time

## Administrator Onboarding

### Initial Setup

#### Step 1: Account Creation
1. **Contact System Administrator**
   - Email: admin@satrf.org
   - Provide your full name and email
   - Specify your role and responsibilities

2. **Receive Admin Access**
   - Admin account created by system admin
   - Receive login credentials
   - Complete security verification

#### Step 2: Admin Dashboard Access
1. **Login to Admin Panel**
   - Go to [satrf.org/admin](https://satrf.org/admin)
   - Use admin credentials
   - Complete two-factor authentication (if enabled)

2. **Dashboard Overview**
   - **System Statistics:** Total users, events, scores
   - **Recent Activity:** Latest registrations and submissions
   - **Pending Actions:** Scores requiring approval
   - **Quick Actions:** Common administrative tasks

### Administrative Functions

#### User Management
1. **View All Users**
   - Go to "Users" in admin menu
   - See complete user list with filters
   - Search by name, email, or club

2. **User Actions**
   - **View Details:** Click on user for full profile
   - **Change Role:** Assign admin, event_scorer, or user roles
   - **Deactivate Account:** Temporarily disable user access
   - **Reset Password:** Generate new password for user

3. **Bulk Operations**
   - Select multiple users
   - Perform bulk role changes
   - Export user data
   - Send bulk communications

#### Score Approval Process
1. **Access Pending Scores**
   - Go to "Scores" → "Pending Approval"
   - View all scores awaiting review
   - Sort by submission date or user

2. **Review Score Details**
   - **User Information:** Name, club, membership type
   - **Event Details:** Competition name and date
   - **Score Data:** Total score, X-count, discipline
   - **Documentation:** Uploaded score sheet
   - **Submission Time:** When score was submitted

3. **Approve or Reject**
   - **Approve Score:**
     - Click "Approve" button
     - Score becomes official
     - User receives confirmation email
     - Leaderboard updates automatically
   
   - **Reject Score:**
     - Click "Reject" button
     - Provide rejection reason
     - User receives notification with reason
     - Score remains in pending status

#### Event Management
1. **Create New Event**
   - Go to "Events" → "Create Event"
   - Fill event details:
     - Title and description
     - Date and time
     - Location and venue
     - Event type and discipline
     - Maximum participants
     - Registration deadline
   - Set event status (draft, open, closed)

2. **Manage Existing Events**
   - **Update Details:** Modify event information
   - **Change Status:** Open/close registration
   - **View Participants:** See registered users
   - **Export Data:** Download participant list

3. **Event Analytics**
   - View registration trends
   - Monitor participant demographics
   - Track event performance
   - Generate reports

#### System Monitoring
1. **Dashboard Statistics**
   - **User Growth:** New registrations over time
   - **Activity Metrics:** Logins, score uploads, event registrations
   - **Performance Data:** System response times
   - **Storage Usage:** Database and file storage

2. **Health Monitoring**
   - **System Status:** Overall platform health
   - **Error Logs:** Recent system errors
   - **Performance Alerts:** Slow response times
   - **Security Events:** Suspicious activity

3. **Backup Management**
   - **Automatic Backups:** Daily database backups
   - **Manual Backups:** On-demand backup creation
   - **Restore Procedures:** Data recovery processes
   - **Backup Verification:** Confirm backup integrity

### Security and Compliance

#### Access Control
- **Role-Based Permissions:** Different access levels for different roles
- **Session Management:** Automatic logout after inactivity
- **Audit Logging:** Track all administrative actions
- **Two-Factor Authentication:** Enhanced security for admin accounts

#### Data Protection
- **GDPR Compliance:** User data protection measures
- **Data Retention:** Automatic cleanup of old data
- **Privacy Controls:** User consent management
- **Data Export:** User data export capabilities

## Troubleshooting

### Common Issues

#### Login Problems
**Issue:** Cannot log in to account
**Solutions:**
1. Check email and password spelling
2. Ensure Caps Lock is off
3. Clear browser cache and cookies
4. Try password reset function
5. Contact support if issues persist

#### Score Upload Issues
**Issue:** Cannot upload score
**Solutions:**
1. Check file size (max 10MB)
2. Verify file format (PDF, JPG, PNG)
3. Ensure all required fields are filled
4. Check internet connection
5. Try refreshing the page

#### Event Registration Problems
**Issue:** Cannot register for event
**Solutions:**
1. Check if event is still open
2. Verify event capacity
3. Ensure you're logged in
4. Check registration deadline
5. Contact event organizer

#### Performance Issues
**Issue:** Slow page loading
**Solutions:**
1. Check internet connection
2. Clear browser cache
3. Try different browser
4. Disable browser extensions
5. Contact technical support
